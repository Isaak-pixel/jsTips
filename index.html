The let Variable

The let keyword is used to declare variables that are block-scoped. They are only useable within the block of code in which they are defined. let is similar to var, but it is subject to the rules of block scoping, whereas var is function-scoped.

The const Variable

The const keyword is used to declare variables that are read-only, meaning that they cannot be reassigned after they are declared. This makes const a useful tool for preventing unintended changes to variables in your code.

Arithmetic Operators

Arithmetic operators perform arithmetic on numbers (literals or variables).

Addition

The addition operator ( + ) adds numbers. For example if x = 5 and y = 8, then x + y = 13

Subtraction

The subtraction operator ( - ) subtracts numbers. For example if x = 8 and y = 5, then x - y = 5

Multiplication

The multiplication operator ( * ) multiplies numbers. For example if x = 5 and y = 8, then x * y = 60

Division

The divison operator ( / ) divides numbers. For example if x = 8 and y = 4, then x / y = 2

Comparison Operators

Comparison operators are used in logical statements to determine equality or difference between variables or values. They return only true and false.

Greater Than ( > )

The greater than operator ( > ) checks if the first number entered is greater than the second number entered. For example: 8 > 4 would return true

Less Than ( < )

The less than operator ( < ) checks if the first number entered is less than the second number entered. For example: 8 < 4 would return false

Equal To ( == )

The equal to operator ( == ) checks if the first number entered is equal to the second number entered. For example: 8 == 4 would return false. The strict equal to (===) returns true only if the two values are equal and of the same type.

Not Equal To ( != )

The not equal to operator ( != ) checks if the first number entered is less than the second number entered. For example: 8 != 4 would return true. The strict not equal to ( !== ) returns true if the two values are equal and are a different type.
